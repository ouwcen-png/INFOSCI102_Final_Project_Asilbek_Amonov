# Final Submission Checklist

## Before submitting on Canvas

- [ ] Run the project locally with `streamlit run app.py`
- [ ] Test at least one loaded headline
- [ ] Test at least one neutral headline
- [ ] Test the comparison tab
- [ ] Confirm `requirements.txt` is included
- [ ] Confirm `README.md` is included
- [ ] Confirm `replication_ai_usage.md` is included
- [ ] Confirm source code is included
- [ ] Record a 4-7 minute demo video
- [ ] Make sure the demo video clearly shows the app working
- [ ] Upload either GitHub link or zip archive to Canvas
- [ ] Upload demo video to Canvas

## After Canvas submission

- [ ] Verify the Canvas submission shows as successfully submitted
- [ ] Open the uploaded files/link once to confirm they work
- [ ] Post demo video under the Discussions thread
- [ ] Post GitHub repo link or code archive under the Discussions thread
- [ ] Make sure classmates can view and rate the project
